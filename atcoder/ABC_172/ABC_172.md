# C - Tsundoku

### 문제

책상 A와 책상 B가 있다. 책상 A에는 수직으로 쌓은 N개의 책이 있고, 책상 B에도 마찬가지로 M개의 책이 쌓여있다.

책상 A에서, 위에서부터 i번째의 책을 읽는 데는 $\ A_i$의 시간이 소요되고, 책상 B에서, 위에서부터 i번째의 책을 읽는 데는 $\ B_i$의 시간이 소요된다.

그리고 다음의 행동을 수행한다.

- 책이 남은 책상을 선택하고 가장 위의 책을 선택하고 읽는다. 그 후 책상에서 해당 책을 제거한다.

K분동안 이 행동을 반복했을 때 최대 몇 개의 책을 읽을 수 있는가?



### 풀이

단순히 두 책상의 맨 위의 책을 읽는데 걸리는 시간을 비교해서 작은 것을 선택하는 방법은 시간이 많이 소요됨 -> 시간 초과

각 책상마다 책을 읽는데 걸리는 시간의 누적합 배열 생성 -> i 부터 N까지 반복하면서 $\ b_j \le K - a_i$를 만족하는 상황에서 i + j가 최대가 되는 경우를 구하면 된다. -> O(N + M)



# D - Sum of Divisors

### 문제

양의 정수 X에 대해, f(x)를 X의 약수의 개수라고 하자.

주어진 양의 정수 N에 대해, $\ \sum_{K=1}^N K \times f(K)$를 구하시오.

### 풀이

일반적으로 약수를 구하는 방법은 $\ O(\sqrt{X})$의 시간이 걸린다. 그러나 모든 K = 1, ... , 10^7에 대해 구하기에는 시간이 충분하지 않다.

생각을 달리해서, 문제를 재정의하면 다음과 같다.

- 양의 정수 j에 대해, g(j)를 N을 초과하지 않는 j의 곱의 합이라고 정의하자. $\ \sum_{j=1}^N g(j)$를 구하시오.

그리고 Y = ⌊N/X⌋라고 할 때, g(X) = X + 2X + ... + YX = Y(Y+1)X/2가 된다. 이 공식을 사용하면 문제는 O(N)의 시간이 소요된다.
